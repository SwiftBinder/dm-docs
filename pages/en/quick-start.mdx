# Quick Start Guide

Get up and running with Document Manager in just 5 minutes.

---

## Step 1: Access the System

Open your web browser and navigate to your Document Manager URL (provided by your administrator).

## Step 2: Log In

1. Enter your **email address**
2. Enter your **password**
3. Click **Sign In**

> **Tip:** If you don't have credentials, contact your system administrator.

After logging in, you'll see the Dashboard — your home base in Document Manager.

{/* SCREENSHOT: dashboard-after-login.png
     Capture the dashboard immediately after login showing:
     - Welcome area with statistics cards
     - Sidebar navigation (collapsed or expanded)
     - Recent Documents section
     - Recent Activity section
     - Browser window: 1400x900
*/}

---

## Step 4: Upload Your First Document

1. Click **Documents** in the sidebar
2. Click the **Upload** button (top right)

{/* SCREENSHOT: documents-page-with-upload-button.png
     Capture the Documents page showing:
     - The page header "Documents"
     - The Upload button prominently visible
     - The document list area (can be empty or with sample data)
     - Browser window: 1400x800
*/}

3. In the upload dialog:
   - Click the upload area or drag a file
   - Enter a **name** for your document
   - Optionally add a **description**
   - Select a **project** (if applicable)
   - Click **Upload**

{/* SCREENSHOT: upload-dialog.png
     Capture the upload dialog showing:
     - File drop zone / upload area
     - Name input field
     - Description input field
     - Project dropdown
     - Upload button
     - Make sure the dialog is clearly visible
*/}

Your document is now uploaded and ready to use!

---

## Step 5: View Your Document

After uploading, click on your document name to view its details.

{/* SCREENSHOT: document-detail-page.png
     Capture a document detail page showing:
     - Document name and description at the top
     - Status badge (Draft, Published, etc.)
     - Tabs: Overview, Versions, Comments
     - Document metadata (tags, project, dates)
     - Download button
     - Browser window: 1400x800
*/}

From here you can:
- **Download** the file
- **Edit** the document metadata
- View **version history**
- Add **comments**

## Quick Reference: Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` / `Cmd+K` | Open command palette (search) |
| `G` then `H` | Go to Home |
| `G` then `D` | Go to Documents |
| `G` then `P` | Go to Projects |

> **Tip:** Press `Ctrl+K` (or `Cmd+K` on Mac) anytime to quickly search for documents, navigate, or perform actions.
