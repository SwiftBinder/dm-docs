# Daily Tasks

Step-by-step instructions for the most common tasks in Document Manager.

---

## Uploading a Document

### Steps

1. Click **Documents** in the left sidebar
2. Click the **Upload** button (top right of the page)

{/* SCREENSHOT: upload-button-location.png
     Capture the Documents page header showing:
     - The "Documents" title
     - The Upload button clearly visible
     - Crop to show just the header area
*/}

3. In the upload dialog that appears:

{/* SCREENSHOT: upload-dialog-filled.png
     Capture the upload dialog showing:
     - A file has been selected (show file name)
     - Name field filled in
     - Description field with example text
     - Project dropdown showing options
     - Upload button at bottom
*/}

   - **Drag and drop** a file onto the upload area, or click to browse
   - Enter a **name** for the document
   - Add a **description** (optional but recommended)
   - Select a **project** from the dropdown
   - Add **tags** to help organize (optional)

4. Click **Upload**

Your document is now uploaded and saved!

> **Tip:** Give your documents clear, descriptive names. Future you will thank you when searching later.

## Finding a Document

### Option 1: Use Search

1. Press `Ctrl+K` (or `Cmd+K` on Mac) to open the command palette
2. Type part of the document name
3. Click the document from the results

{/* SCREENSHOT: command-palette-search.png
     Capture the command palette showing:
     - The search input at top
     - Search results below with document names
     - Keyboard shortcut hint visible
*/}

### Option 2: Browse the Document List

1. Click **Documents** in the sidebar
2. Use the filters at the top:
   - **Search box** — Type keywords
   - **Status filter** — Show only Draft, Published, etc.
   - **Project filter** — Show documents from a specific project

{/* SCREENSHOT: documents-filters.png
     Capture the Documents page filter area showing:
     - Search input field
     - Status dropdown filter
     - Project dropdown filter
     - Clear, well-composed view
*/}

### Option 3: Check Recent Documents

Your most recently accessed documents appear on the Dashboard under "Recent Documents."

## Viewing a Document

1. Find the document (using any method above)
2. Click on the document name

The document detail page shows:

{/* SCREENSHOT: document-detail-overview.png
     Capture a document detail page showing:
     - Document name at top
     - Status badge
     - Description
     - Metadata sidebar (project, tags, dates)
     - Download button

- **Name and Description** — Basic information

## Downloading a Document

### Download Current Version

1. Open the document detail page

{/* SCREENSHOT: download-button.png

## Adding a Comment

1. Open the document detail page
2. Click the **Comments** tab
3. Type your comment in the text box
4. Click **Post**

{/* SCREENSHOT: add-comment.png
     Capture the Comments tab showing:
     - The comment input area
     - Post button
     - Existing comments (if any) for context
*/}

> **Tip:** Use comments to ask questions, give feedback, or discuss changes with your team.

---

## Replying to a Comment

1. Find the comment you want to reply to
2. Click **Reply**
3. Type your response
4. Click **Post**

Replies are grouped under the original comment as a thread.

## Changing Document Status

If you have permission, you can update a document's status:

1. Open the document detail page
2. Click the **Edit** button (or click on the metadata area)
3. Change the **Status** dropdown
4. Save your changes

{/* SCREENSHOT: edit-status.png
     Capture the document edit mode showing:
     - Status dropdown open or visible
     - Available status options
*/}

| Status | When to Use |
|--------|-------------|
| **Draft** | Document is being worked on |
| **Published** | Document is finalized and ready for use |
| **Archived** | Document is no longer active but should be kept |

## Uploading a New Version

When you have an updated version of a document:

1. Open the document detail page
2. Click the **Versions** tab
3. Click **Upload New Version**

{/* SCREENSHOT: upload-version-button.png
     Capture the Versions tab showing:
     - The "Upload New Version" button
     - Existing version list for context
*/}

4. In the dialog:
   - Upload the new file
   - Add a **change summary** (what changed in this version)
   - Click **Upload**

{/* SCREENSHOT: version-upload-dialog.png
     Capture the version upload dialog showing:
     - File upload area
     - Change summary field
     - Upload button
*/}

The old version is preserved, and your new version becomes the current version.

## Organizing with Tags

Tags help you categorize and find documents.

### Adding Tags to a Document

1. Open the document detail page
2. Click **Edit**
3. Type tag names in the **Tags** field
4. Save your changes

{/* SCREENSHOT: add-tags.png
     Capture the tag input area showing:
     - Tags field with example tags
     - How to add/remove tags
*/}

### Finding Documents by Tag

1. Click **Tags** in the sidebar
2. Click on a tag to see all documents with that tag

## Quick Actions Summary

| Task | How |
|------|-----|
| Upload document | Documents → Upload button |
| Find document | Press `Ctrl+K` and search |
| Download | Document page → Download button |
| Add comment | Document page → Comments tab → Post |
| New version | Document page → Versions tab → Upload New Version |
| Change status | Document page → Edit → Status dropdown |
| Add tags | Document page → Edit → Tags field |
