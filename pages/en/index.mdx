## What is Document Manager?

Document Manager is an enterprise-grade document management system designed for teams who need:

- **Centralized Storage** — All your documents in one secure location
- **Version Control** — Track every change with complete history
- **Team Collaboration** — Comments, discussions, and shared workflows
- **Project Organization** — Group documents by projects and teams
- **Access Control** — Role-based permissions for security

## User Roles at a Glance

Document Manager uses four roles to control access:

| Role | Can Do |
|------|--------|
| **Viewer** | View documents and comments |
| **User** | Create and edit documents, add comments |
| **Project Manager** | Manage projects and teams |
| **Admin** | Full system access including user management |

> **Note:** Your role determines which features you can access. Some menu items may not appear if you don't have permission.
