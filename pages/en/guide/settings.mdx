## Profile Settings

### Your Profile Information

The Profile section displays and allows editing of your personal information.

{/* SCREENSHOT: profile-section.png
     Capture the Profile section showing:

     - Username field
     - Email field (read-only)
     - Avatar if shown

### Editable Fields

| Field | Description | Can Edit? |
|-------|-------------|-----------|
| **Display Name** | Name shown to others | Yes |
| **Username** | Your login identifier | Sometimes |
| **Email** | Your email address | Usually No* |

*Email changes typically require administrator assistance.

### Updating Your Display Name

1. Go to Settings
2. Find the Display Name field
3. Enter your preferred name
4. Click **Save** or it auto-saves

## Templates & Stages

The Settings page includes management for project templates and workflow stages.

### Templates Tab

Access project template management from the Settings page:

{/* SCREENSHOT: settings-templates-tab.png
     Capture the Templates tab showing:
     - System Templates section
     - My Templates section
     - Template cards with name and category
     - Create, Edit, Delete actions
*/}

#### System Templates

Pre-configured templates provided by your organization:
- **View** — See template details and stages
- **Duplicate** — Create your own version to customize

#### My Templates

Templates you've created:
- **Create** — Build new templates from scratch
- **Edit** — Modify existing templates
- **Delete** — Remove templates you no longer need

#### Creating a Template

1. Click **Create Template** (or Duplicate a system template)
2. Fill in template details:

| Field | Description |
|-------|-------------|
| **Name** | Template title |
| **Description** | What the template is for |
| **Category** | Grouping (e.g., Documentation, Approval) |
| **Stages** | Workflow stages documents will go through |

3. Configure stages (see Stages Tab below)
4. Save

### Stages Tab

Configure workflow stages for project templates:

{/* SCREENSHOT: settings-stages-tab.png
     Capture the Stages tab showing:
     - Stage template list
     - Stage names with colors
     - Order/sequence
     - Edit and delete actions
*/}

#### Stage Configuration

Each stage has:

| Property | Description |
|----------|-------------|
| **Name** | Stage title (e.g., "Draft", "Review", "Published") |
| **Color** | Visual indicator in Kanban and badges |
| **Order** | Sequence in the workflow |
| **Is Initial** | First stage for new documents |
| **Is Final** | End stage (no further transitions) |
| **Allowed Transitions** | Which stages can follow this one |

#### Creating Stages

1. Go to Settings → Stages
2. Click **Create Stage Template**
3. Add stages in order
4. Define allowed transitions between stages
5. Save

> **Tip:** Start simple with 3-4 stages (Draft → Review → Published) and add complexity only when needed.

## Notification Settings

Control how Document Manager notifies you of activity.

{/* SCREENSHOT: notification-settings.png
     Capture the Notification settings showing:
     - Enable/disable toggles
     - Notification categories
     - Save button if applicable
*/}

### Notification Types

| Category | What It Covers |
|----------|----------------|
| **Documents** | New documents, updates |
| **Comments** | New comments, replies |
| **Projects** | Project changes, assignments |
| **System** | Important announcements |

### Enabling/Disabling

1. Go to Settings
2. Find Notifications section
3. Toggle each category on/off
4. Save if required

### Notification Methods

Depending on configuration:
- **In-app** — Bell icon in top bar
- **Email** — Sent to your email address
- **Both** — Combination

## Security Settings

### Changing Your Password

1. Go to Settings
2. Find Security section
3. Click **Change Password**

{/* SCREENSHOT: change-password-form.png
     Capture the Change Password dialog showing:
     - Current password field
     - New password field
     - Confirm password field
     - Save button
*/}

4. Enter your **current password**
5. Enter your **new password**
6. Confirm the new password
7. Click **Save** or **Change Password**

### Password Requirements

| Requirement | Minimum |
|-------------|---------|
| Length | 8 characters |
| Complexity | Varies by configuration |

### Password Tips

- Use a unique password not used elsewhere
- Consider using a password manager
- Change password if you suspect compromise

### Session Security

Your active sessions are managed automatically:
- Tokens refresh in the background
- Sessions expire after inactivity
- Logging out clears the session

## Preferences That Sync

These preferences are saved to your account:

| Preference | Where Saved |
|------------|-------------|
| Theme | Account |
| Default view | Account |
| Notification settings | Account |

This means:
- Settings follow you across devices
- Logging in elsewhere uses your preferences

## Preferences Saved Locally

Some preferences are saved in your browser:

| Preference | Where Saved |
|------------|-------------|
| Sidebar collapsed/expanded | Browser |
| Table column widths | Browser |

This means:
- Different on each device/browser
- Cleared if you clear browser data

## Account Management

### Requesting Role Change

If you need different permissions:

1. Identify what access you need
2. Contact your project manager or administrator
3. Explain the business reason
4. Wait for approval and update

### Account Deactivation

If you need to deactivate your account:

1. Contact your administrator
2. They can deactivate your account
3. Your data is preserved but access is revoked

### Account Deletion

For complete account removal:

1. Contact your administrator
2. Discuss data retention requirements
3. Administrator processes the request

## Troubleshooting Settings

### Changes Not Saving

**Try:**
- Click Save button explicitly
- Refresh the page and try again
- Check for error messages
- Clear browser cache

### Theme Not Applying

**Check:**
- Browser supports theme (modern browser)
- JavaScript is enabled
- Try a different theme, then switch back

### Password Change Failed

**Possible causes:**
- Current password incorrect
- New password doesn't meet requirements
- Network issue — try again

### Notifications Not Working

**Check:**
- Notifications are enabled in settings
- Browser allows notifications
- Check spam folder for emails
