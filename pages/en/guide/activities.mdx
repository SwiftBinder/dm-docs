## Understanding Activities

### What Gets Tracked

| Category | Events |
|----------|--------|
| **Documents** | Created, updated, deleted, downloaded |
| **Versions** | Uploaded, rolled back, downloaded |
| **Comments** | Added, edited, resolved, deleted |
| **Projects** | Created, updated, archived, deleted |
| **Users** | Logged in, profile updated |
| **Teams** | Created, members added/removed |

### Why Activity Tracking Matters

- **Audit compliance** ‚Äî Know who did what and when
- **Collaboration awareness** ‚Äî See team activity
- **Troubleshooting** ‚Äî Understand what changed
- **Security** ‚Äî Detect unusual activity

## Notifications vs Activities

### How They Differ

| Aspect | Notifications | Activities |
|--------|---------------|------------|
| **Purpose** | Alert you to important events | Complete record |
| **Scope** | Events relevant to you | All system events |
| **Read status** | Can mark as read | No read status |
| **Persistence** | May be cleared | Permanent record |

### When to Use Each

- **Notifications** ‚Äî What needs your attention now
- **Activities** ‚Äî Historical record of what happened

## Troubleshooting

### Activities Not Loading

**Try:**
- Refresh the page
- Clear filters
- Check internet connection

### Missing Activities

**Check:**
- Date filter range
- Resource type filter
- User permissions

### Activity Details Unavailable

**Possible causes:**
- Resource was deleted
- Insufficient permissions
- Data retention expired

## Technical Details

### What Gets Logged

Each activity records:

| Field | Description |
|-------|-------------|
| Event type | Type of action |
| Resource type | Document, project, etc. |
| Resource ID | Specific item affected |
| User ID | Who performed action |
| Timestamp | When it happened |
| IP address | Client location |
| Event data | Additional details |

### API Access

Developers can access activities:

```
GET /api/v1/activities/recent
GET /api/v1/activities/me
GET /api/v1/activities/{id}
```

### Activity Retention

Check with your administrator for:
- How long activities are kept
- Archival policies
- Export capabilities
